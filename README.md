# thebestfrozenpage
frozenforivy

A phone-friendly webpage with Ivy's Frozen bedtime story and messages from Frozen friends.

## Features
- Responsive design for mobile and desktop
- Frozen-themed story and characters
- Audio player for songs

## Publishing
This site is published on GitHub Pages at: https://scorpiogirl10251990.github.io/thebestfrozenpage/

To enable GitHub Pages:
1. Go to the repository on GitHub
2. Settings > Pages
3. Source: Deploy from a branch
4. Branch: main, Folder: /(root)
5. Save

The site should be live shortly.
